# M3SDK Document List

This page provides a list of SDK documents for M3 devices. Each SDK is available in both Korean and
English.

## KeyTool SDK

These are the SDK documents related to the KeyTool application.

* **FN Key Control SDK**: Guides on how to control the state of the FN key (enable, disable, lock).
    * [Korean](./keytool/sl-series-fn-control-sdk-korean.md)
    * [English](./keytool/sl-series-fn-control-sdk-english.md)

* **Key Setting SDK**: Explains how to remap physical keys to different functions or change Wake-up
  settings.
    * [Korean](./keytool/sl-series-key-setting-sdk-korean.md)
    * [English](./keytool/sl-series-key-setting-sdk-english.md)

* **Wakeup Control SDK(Only for SL20P) (Deprecated)**: A legacy SDK for controlling the Wake-up
  function of specific keys. (For new development, it is recommended to use the 'Key Setting SDK'.)
    * [Korean](./keytool/sl20p-wakeup-control-sdk-korean.md)
    * [English](./keytool/sl20p-wakeup-control-sdk-english.md)
